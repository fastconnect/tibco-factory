FCUnit for TIBCO BusinessWorks

Automatic TIBCO BusinessWorks unit testing

FCUnit is an automatic tool to execute Unit Tests on TIBCO BusinessWorks projects.

The Unit Tests can be:

  • defined directly in TIBCO BusinessWorks processes
  • data-driven, with input and expected XML
